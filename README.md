# TRG TTag 产线监控工具

实时监听 TRG 测温基站上报的 **55 AA 协议帧**，解析 TTag 无线温度标签数据，支持多层过滤与 CSV 导出。

## 快速开始

```bash
# 安装依赖（纯标准库，无需 pip install）
# 直接运行
python ttag_monitor.py

# 默认监听 0.0.0.0:20226，仅显示 T 型标签 (0x00)
```

## 命令行参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--ip` | `0.0.0.0` | 监听 IP |
| `--port` | `20226` | 监听端口 |
| `--type` | `0x00` | 标签类型过滤，逗号分隔 |
| `--rule NAME START END` | — | ID 区间过滤规则（可多次） |
| `--export PATH` | — | 导出 CSV 文件路径 |

### 示例

```bash
# 监听多个标签类型
python ttag_monitor.py --type 0x00,0x03

# 按 ID 区间过滤
python ttag_monitor.py --rule A产线 175000 175999 --rule B产线 200000 200999

# 导出到 CSV
python ttag_monitor.py --export data.csv
```

## 交互命令

运行时支持以下交互命令：

| 命令 | 说明 |
|------|------|
| `rule add <名称> <起始ID> <结束ID>` | 添加 ID 区间过滤 |
| `rule del <名称>` | 删除规则 |
| `rule list` | 列出所有规则 |
| `stats` | 显示统计 |
| `export <文件名>` | 开始/切换 CSV 导出 |
| `export off` | 停止导出 |
| `quit` | 退出 |

## 协议概述

### 帧结构

```
 55 AA  | 长度  | 基站ID | 功能码 | SN | 标签数 | 标签块 ×N | 校验
 2B     | 2B LE | 2B LE  | 1B     | 1B | 1B     | 9B×N      | 1B
```

### 标签块 (每标签 9 字节)

| 偏移 | 长度 | 内容 |
|------|------|------|
| +0 | 1B | RSSI 信号强度 |
| +1 | 1B | 标签类型 |
| +2 | 3B | 标签 ID（大端序） |
| +5 | 2B | ADC 值（小端序） |
| +7 | 2B | 保留 |

- ADC = `0xFFFF` 表示低电量
- 标签类型 `0x00` = T 型温度标签

## 过滤流程

```
接收帧 → 解析标签 → [1] 标签类型过滤 → [2] ID 区间过滤 → [3] ADC 范围过滤 → 显示/导出
```
